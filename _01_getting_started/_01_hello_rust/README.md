## Install

```
❯ brew install rustup
```

## Compile
- Compiling and Running are Seperate Steps in Rust
    - Before running we must compile it using the rust compiler by entering `rustc` command and passing the name of the soruce file, like this:
```
❯ rustc main.rs
```

## Run
```
❯ ./main
```
